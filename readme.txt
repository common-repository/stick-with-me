Make elements sticky on the top of the page (modified version of Sticky Element plugin by Heckenberg).

== Description ==
Simple configuration for making your elements sticky? Fortunately, this plugin offers the features you want!

If you have any suggestions/comments, please let me know. Your opinions are highly appreciated.

== Installation ==
1. Upload 'sticky-element' directory to '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
* Q1. What is z-index?
* A1. The z-index property specifies the stack order of an element. The higher the value, the higher the chance your element will appear in top (or front) of all the elements
